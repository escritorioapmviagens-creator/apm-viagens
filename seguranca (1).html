A nossa abordagem à segurança

Na APM Viagens, a segurança dos teus dados e das tuas transações é uma prioridade absoluta. Implementamos múltiplas camadas de proteção para garantir que a tua informação está sempre segura.

🔒

Encriptação HTTPS

Toda a comunicação entre o teu dispositivo e os nossos servidores é encriptada com TLS 1.3, o protocolo mais seguro disponível.

🛡️

Autenticação Segura

Sistema de autenticação por link mágico — sem passwords armazenadas. Cada acesso é verificado de forma única e temporária.

🏦

Dados Financeiros

Não armazenamos dados de cartões de crédito. Os pagamentos são processados por referência Multibanco ou transferência bancária direta.

🌐

Infraestrutura

Os dados são armazenados na plataforma Supabase, com servidores na União Europeia, em conformidade com o RGPD.

👁️

Controlo de Acessos

Sistema de permissões por funções — cada consultor acede apenas aos seus próprios dados. Dados de terceiros são inacessíveis.

💾

Backups

Backups automáticos diários da base de dados. Os teus dados nunca se perdem.

Pagamentos Seguros

A APM Viagens aceita pagamentos por:

  • Referência Multibanco — sistema bancário português certificado, sem exposição de dados financeiros
  • Transferência Bancária — para conta bancária verificada da APM Viagens
⚠️ Atenção: A APM Viagens nunca solicita dados de cartão de crédito por email, WhatsApp ou telefone. Se receberes um pedido deste tipo, contacta-nos imediatamente.

Proteção de Dados Pessoais

  • Os teus dados são tratados em conformidade com o RGPD (Regulamento UE 2016/679)
  • Acesso restrito a colaboradores com necessidade operacional justificada
  • Registos de acesso monitorizados e auditados
  • Eliminação segura de dados após o período de retenção legal

As tuas responsabilidades

Para manteres a tua conta segura, recomendamos:

  • Não partilhes os links de acesso enviados por email com terceiros
  • Usa um endereço de email seguro e com acesso exclusivo teu
  • Faz logout após usar a área do cliente em dispositivos partilhados
  • Notifica-nos imediatamente se suspeitares de acesso não autorizado

Vulnerabilidades e Incidentes

Se descobrires uma vulnerabilidade de segurança no nosso sistema, agradecemos que nos contactes de forma responsável antes de qualquer divulgação pública:
